## 1. 模組概述

### 1.1 模組定位

M16 回測系統是 Layer 5 應用層模組，提供策略回測與績效驗證功能。系統接收來自 M11 量化策略模組的策略定義與 M13 信號引擎的統一信號，結合 M06 的歷史價格數據，模擬策略在歷史行情中的表現，計算各項績效指標，幫助用戶驗證投資策略的有效性。

### 1.2 核心價值

| 價值面向 | 說明 |
|---------|------|
| **策略驗證** | 在實際投入資金前驗證策略有效性 |

**輸入參數**：

```json

  "strategyId": "STR_001",              // M11 策略 ID（可選）
  "signalSource": "STRATEGY",           // STRATEGY | SIGNAL | CUSTOM
  "targetStocks": {
    "mode": "SPECIFIC",                 // SPECIFIC | WATCHLIST | FILTER
    "stockIds": ["2330", "2317"],
    "watchlistId": null,
    "filterConditions": null
  },
  "period": {
    "startDate": "2023-01-01",
    "endDate": "2025-12-31"
  },
  "initialCapital": 1000000,            // 初始資金
  "tradingSettings": {
    "positionSizing": "FIXED_AMOUNT",   // 部位大小方式
    "fixedAmount": 100000,              // 固定金額
    "maxPositions": 5,                  // 最大持股數
    "allowShort": false,                // 是否允許放空
    "tradingCost": {
      "commissionRate": 0.001425,       // 手續費率
      "taxRate": 0.003,                 // 交易稅率
      "slippage": 0.001                 // 滑價

  "exitRules": {
    "stopLoss": 0.07,                   // 停損 7%
    "takeProfit": 0.15,                 // 停利 15%
```

#### F-M16-003 績效計算

**功能說明**：計算回測結果的各項績效指標。

**績效指標定義**：

| 指標 | 公式 | 說明 |
|-----|------|------|
| **總報酬率** | (期末淨值 - 初始資金) / 初始資金 × 100% | 整體獲利率 |
| **年化報酬率** | (1 + 總報酬率)^(252/交易日數) - 1 | 年化後的報酬 |
| **夏普比率** | (年化報酬 - 無風險利率) / 年化波動度 | 風險調整後報酬 |
| **索提諾比率** | (年化報酬 - 無風險利率) / 下行波動度 | 下行風險調整報酬 |
| **最大回撤** | max((峰值 - 谷底) / 峰值) | 最大虧損幅度 |
| **勝率** | 獲利交易數 / 總交易數 × 100% | 獲利機率 |
| **盈虧比** | 平均獲利 / 平均虧損 | 賺賠比例 |
| **獲利因子** | 總獲利 / 總虧損 | 整體盈虧比 |
| **卡瑪比率** | 年化報酬 / 最大回撤 | 報酬回撤比 |
| **交易次數** | 買進 + 賣出次數 | 總交易頻率 |
| **平均持有天數** | 總持有天數 / 交易次數 | 平均持股時間 |
| **週轉率** | 總交易金額 / 平均資產 | 資金使用效率 |

#### F-M16-008 基準比較

**功能說明**：將回測結果與市場基準（如加權指數）進行比較。

**支援基準**：

| 基準代碼 | 名稱 | 說明 |
|---------|------|------|
| TAIEX | 加權指數 | 台灣加權股價指數 |
| TPEx | 櫃買指數 | 櫃檯買賣指數 |
| TECH | 電子類指數 | 電子類股指數 |
| FIN | 金融類指數 | 金融類股指數 |

**Alpha/Beta 計算**：

- **Alpha** = 策略報酬 - (無風險利率 + Beta × (基準報酬 - 無風險利率))
- **Beta** = Cov(策略報酬, 基準報酬) / Var(基準報酬)

### 2.4 P2 次要功能

### 3.2 元件職責

| 元件 | 職責 |
|-----|------|
| **DataLoader** | 從 M06 載入歷史價格，從 M07 載入技術指標 |
| **SignalScanner** | 掃描 M11 策略條件或 M13 統一信號 |
| **TradeExecutor** | 模擬交易執行，計算手續費、交易稅、滑價 |
| **Portfolio** | 維護虛擬帳戶：持倉、現金、淨值曲線 |
| **RiskChecker** | 檢查出場條件：停損、停利、移動停損、持有天數 |
| **MetricsCalc** | 計算績效指標：報酬率、夏普比率、最大回撤等 |
