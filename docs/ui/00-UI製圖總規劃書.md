## 1. 專案概述

### 1.1 技術棧

| 項目 | 技術選擇 | 說明 |
|-----|---------|------|
| 框架 | Vue 3 | Composition API |
| 語言 | TypeScript | 強型別 |
| UI 庫 | Element Plus / Naive UI | 企業級元件庫 |
| 圖表 | ECharts / TradingView | 股票圖表 |
| 狀態管理 | Pinia | Vue 3 官方推薦 |
| 路由 | Vue Router 4 | |
| HTTP | Axios | API 呼叫 |
| 建構 | Vite | 快速開發 |

### 1.2 設計原則

| 原則 | 說明 |
|-----|------|
| **一致性** | 所有模組使用統一的設計語言 |
| **效率優先** | 投資人需要快速獲取資訊 |
| **資料密度** | 在有限空間展示最大資訊量 |
| **響應式** | 支援桌面與平板 |
| **深色模式** | 投資人偏好深色介面 |

### 1.3 目標用戶

| 用戶類型 | 主要需求 | 常用功能 |
|---------|---------|---------|
| 主動投資人 | 選股、技術分析 | M14 選股、M07 技術指標 |
| 法人研究員 | 基本面、籌碼分析 | M08 基本面、M09 籌碼 |
| 量化交易者 | 策略回測、風控 | M11 策略、M16 回測 |
| 一般散戶 | 警報通知、持股追蹤 | M15 警報、M18 投資組合 |

## 2. 系統導航架構

### 2.1 主導航結構

```
┌─────────────────────────────────────────────────────────────────┐
│  LOGO   │ 市場總覽 │ 個股分析 │ 選股中心 │ 策略回測 │ 我的投資 │ ⚙️ │
└─────────────────────────────────────────────────────────────────┘

主導航 (Top Nav)
├── 市場總覽 (Dashboard)
│   ├── 大盤指數
│   ├── 類股表現
│   └── 熱門排行
│
├── 個股分析 (Stock Analysis)
│   ├── 技術分析 (M07)
│   ├── 基本面分析 (M08)
│   ├── 籌碼分析 (M09)
│   └── 型態辨識 (M10)
│
├── 選股中心 (Stock Screening)
│   ├── 多條件選股 (M14)
│   ├── 預設策略
│   └── 信號總覽 (M13)
│
├── 策略回測 (Backtesting)
│   ├── 策略建立 (M11)
│   ├── 回測執行 (M16)
│   └── 績效報告
│
├── 我的投資 (My Investment)
│   ├── 投資組合 (M18)
│   ├── 警報設定 (M15)
│   └── 風險管理 (M17)
│
└── 系統管理 (Admin)
    ├── 資料管理 (M06)
    ├── Job 監控
    └── 系統設定
```

### 2.2 導航層級定義

| 層級 | 說明 | 範例 |
|-----|------|------|
| L1 主導航 | 頂部固定導航 | 市場總覽、個股分析... |
| L2 子導航 | 左側選單或 Tab | 技術分析、基本面分析... |
| L3 頁面內容 | 主內容區 | 指標圖表、數據表格... |
| L4 詳情/Modal | 彈窗或抽屜 | 指標說明、設定面板... |

### 2.3 URL 路由規劃

```
/                           # 首頁 (市場總覽)
/dashboard                  # 市場總覽
  /dashboard/index          # 大盤指數
  /dashboard/sector         # 類股表現
  /dashboard/ranking        # 熱門排行

/stock/:stockId             # 個股分析
  /stock/:stockId/technical # 技術分析 (M07)
  /stock/:stockId/fundamental # 基本面 (M08)
  /stock/:stockId/chip      # 籌碼分析 (M09)
  /stock/:stockId/pattern   # 型態辨識 (M10)

/screener                   # 選股中心
  /screener/custom          # 多條件選股 (M14)
  /screener/preset          # 預設策略
  /screener/signals         # 信號總覽 (M13)

/strategy                   # 策略回測
  /strategy/builder         # 策略建立 (M11)
  /strategy/backtest        # 回測執行 (M16)
  /strategy/report/:id      # 績效報告

  /portfolio/holdings       # 投資組合 (M18)

/admin                      # 系統管理
  /admin/data               # 資料管理 (M06)
  /admin/jobs               # Job 監控
  /admin/settings           # 系統設定
```

## 3. 設計系統規範

### 3.1 色彩系統

#### 主色調（深色模式為主）

```css
/* 品牌色 */
--primary: #3B82F6;          /* 藍色 - 主要操作 */
--primary-hover: #2563EB;

/* 語義色 */
--success: #10B981;          /* 綠色 - 漲/買進 */
--danger: #EF4444;           /* 紅色 - 跌/賣出 */
--warning: #F59E0B;          /* 橙色 - 警告 */
--info: #6366F1;             /* 紫色 - 資訊 */

/* 背景色（深色模式） */
--bg-primary: #0F172A;       /* 主背景 */
--bg-secondary: #1E293B;     /* 卡片背景 */
--bg-tertiary: #334155;      /* 輸入框背景 */

/* 文字色 */
--text-primary: #F8FAFC;     /* 主要文字 */
--text-secondary: #94A3B8;   /* 次要文字 */
--text-muted: #64748B;       /* 輔助文字 */

/* 邊框色 */
--border: #334155;
--border-hover: #475569;
```

#### 股票漲跌色

```css
/* 台股慣例：紅漲綠跌 */
--stock-up: #EF4444;         /* 漲 - 紅色 */
--stock-down: #10B981;       /* 跌 - 綠色 */
--stock-flat: #6B7280;       /* 平盤 - 灰色 */

/* 漲跌背景（淺色） */
--stock-up-bg: rgba(239, 68, 68, 0.1);
--stock-down-bg: rgba(16, 185, 129, 0.1);
```

### 3.2 字型系統

```css
/* 字型家族 */
--font-sans: 'Inter', 'Noto Sans TC', sans-serif;
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;

/* 字型大小 */
--text-xs: 0.75rem;    /* 12px - 輔助資訊 */
--text-sm: 0.875rem;   /* 14px - 次要內容 */
--text-base: 1rem;     /* 16px - 主要內容 */
--text-lg: 1.125rem;   /* 18px - 小標題 */
--text-xl: 1.25rem;    /* 20px - 標題 */
--text-2xl: 1.5rem;    /* 24px - 大標題 */
--text-3xl: 1.875rem;  /* 30px - 頁面標題 */

/* 數字專用（等寬） */
.number {
  font-family: var(--font-mono);
  font-feature-settings: 'tnum';  /* 表格數字 */
}
```

/* 基準單位: 4px */

### 3.4 元件規範

#### 按鈕

| 類型 | 用途 | 樣式 |
|-----|------|------|
| Primary | 主要操作（儲存、確認） | 實心藍色 |
| Secondary | 次要操作（取消） | 邊框藍色 |
| Danger | 危險操作（刪除） | 實心紅色 |
| Ghost | 輔助操作 | 透明背景 |

#### 卡片

```css
.card {
  background: var(--bg-secondary);
  border: 1px solid var(--border);
  border-radius: 8px;
  padding: var(--space-4);
}
```

#### 表格

```css
.table {
  /* 斑馬紋 */
  tr:nth-child(even) {
    background: var(--bg-tertiary);
  }
  /* 懸停 */
  tr:hover {
    background: rgba(59, 130, 246, 0.1);
  }
  /* 數字右對齊 */
  td.number {
    text-align: right;
    font-family: var(--font-mono);
  }
}
```
